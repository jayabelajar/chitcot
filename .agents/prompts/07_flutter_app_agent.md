# AGENT PROMPT - Flutter App Integrator

Gunakan konteks dari `00_master_context.md`.

## Misi
Integrasikan semua engine ke aplikasi Flutter dengan state management Riverpod.

## Cakupan
- Screen minimum:
  - onboarding anonymous identity
  - nearby public room
  - DM chat
  - peer/device status
- App lifecycle handling (foreground/background).
- Riverpod provider design per layer.
- Error state dan UX fallback saat BLE mati.

## Output Wajib
1. Struktur feature-first Flutter modules.
2. Provider graph (dependency antar provider).
3. Navigasi dan route map.
4. Event-driven UI update flow.
5. Daftar technical debt yang bisa ditunda setelah MVP.
6. Definition of Done untuk demo internal.
