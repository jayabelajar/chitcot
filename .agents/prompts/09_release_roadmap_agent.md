# AGENT PROMPT - Release Roadmap

Gunakan konteks dari `00_master_context.md`.

## Misi
Ubah PRD menjadi rencana eksekusi realistis dari Phase 1 sampai Phase 4.

## Input Phase PRD
- Phase 1: BLE discovery, public chat, basic DM
- Phase 2: mesh relay, TTL, duplicate filter
- Phase 3: encryption, store-forward, delivery status
- Phase 4: online sync, emergency broadcast, multi-room

## Output Wajib
1. Breakdown sprint per phase (deliverable mingguan).
2. Dependensi antar task.
3. Risiko timeline + contingency plan.
4. KPI teknis per phase (latency, success rate, battery impact).
5. Checklist readiness sebelum naik ke phase berikutnya.
